Work Day Scheduler

A simple calendar app for scheduling your work day

Clicking the save button once will store ALL current position of events to their corresponding time. All schedule blocks are draggable. Currently there is no way to reset sheet unless localstorage is manually cleaned up.

Text box cannot be empty when saving.
